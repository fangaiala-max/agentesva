Programar ha dejado de ser solo para programadores. Una nueva ola de herramientas con IA deja **crear o prototipar software describiendo en español lo que quieres**, sin escribir todo el código a mano.

**[Cursor](/herramienta/cursor)** es un editor de código con IA en el centro: edita varios archivos por instrucción, entiende tu proyecto y aplica cambios. Tiene plan gratuito para empezar.

**[v0](/herramienta/v0)**, de Vercel, genera **interfaces web** (componentes React/Tailwind) a partir de una descripción o una imagen, listas para copiar a tu proyecto.

**[Bolt](/herramienta/bolt)** construye y ejecuta **aplicaciones completas** en el navegador a partir de una descripción, con vista previa y despliegue.

Y para quien ya programa, **[GitHub Copilot](/herramienta/github-copilot)** sigue siendo el copiloto de referencia dentro del editor.

¿Para quién? Desde equipos sin perfil técnico que quieren prototipar una idea, hasta desarrolladores que van más rápido. Fuente: páginas oficiales de cada herramienta.
